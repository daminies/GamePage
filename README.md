# GamePage

## 기획의도 
- 코로나로 인한 게임의 관심 ↑
- 한국에는 관련 사이트가 부재

## 기능
- 게시판 기능 
  (DB연동 글 등록, 수정, 삭제 기능) 
- 장르별, 가격대별 안내 서비스
- 게임에 대한 세부적인 정보 제공
- 상단바 제공을 통한 편리한 화면이동
- 검색기능
- 크롤링을 이용한 게임 할인 정보 제공
